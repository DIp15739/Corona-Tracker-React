# COVID-19 Tracker

I create a COVID-19 Tracker/Stats Application. I'm going to use React with the addition of Charts.js, Material UI, and many different modules.

API used: https://covid19.mathdro.id/api

Setup:

- run `npm i`
- run `npm start`
